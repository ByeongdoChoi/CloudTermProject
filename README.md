# CloudTermProject
1.환경
-운영체제: 우분투 20.04

-자바: 버전8 이상 설치

-메이븐 설치

- ~/.aws/config 에서

[default]
region = YOUR_REGION 
입력

- ~/.aws/credentials 에서

[default]

aws_access_key_id = YOUR_AWS_ACCESS_KEY_ID

aws_secret_access_key = YOUR_AWS_SECRET_ACCESS_KEY

입력 

2.실행

-master 브런치의 프로젝트를 다운 받은 후 프로젝트 실행
